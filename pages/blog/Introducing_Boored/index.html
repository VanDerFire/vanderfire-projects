Introducing: Boored.py

A simple booru media download library

June 9 2022

With Boored.py you can extract media from various imageboards with only one library.

This library interacts with the API of these pages automatically using JSON and urllib for get the data of the APIs

Boored.py 1.0 List of supported sites

  • rule34.xxx
  • e621.net
  • gelbooru.com
  • realbooru.net
  • safebooru.org
  • yande.re

Check the releases in the official github repository.
